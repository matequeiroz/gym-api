# Gym API

API para gerenciamento de academias.

## Começando

### Pré-requisitos

- Node.js (v18 ou superior)
- Yarn package manager

### Instalação

```bash
yarn install
```

### Executando a Aplicação

Modo de desenvolvimento:
```bash
yarn start:dev
```

Modo de produção:
```bash
yarn build
yarn start:prod
```

## Endpoints da API

- `GET /health` - Endpoint de status da aplicação

## Scripts

- `yarn dev` - Iniciar servidor de desenvolvimento com auto-reload
- `yarn start:dev` - Iniciar servidor de desenvolvimento
- `yarn build` - Compilar a aplicação para produção
- `yarn start:prod` - Iniciar servidor de produção
